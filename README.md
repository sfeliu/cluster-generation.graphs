# algo3TP2


## Como compilar desde la terminal
1. `mkdir build && cd build*`
2. `cmake ..`
3. `make`

Para recompilar, simplemente correr `make` de nuevo

## Para correr el primer punto
- ./tp 0 (012) 
    0 = Kruskal
    1 = Kruskal PC
    2 = Prim

## Para correr el segundo punto
- ./tp 1 (01) 
    0 = Bellman-Ford
    1 = Floyd-Warshall
